# ORBITA – PRODUCT REQUIREMENTS DOCUMENT (PRD)

## Product Scope

Orbita is built as a **CLI-first system** with a shared API that later powers Web and App experiences.

Orbita consists of:

- Orbita Core (CLI + API)
- Modular add-ons (Orbits)
- Interfaces:
  - CLI (primary)
  - MCP (AI integrations)
  - Web App (Phase 3)
  - PWA / Mobile (Phase 3)

## Target Users

### Primary

- Knowledge workers (25–45)
- Engineers, founders, consultants
- ADHD-friendly productivity seekers

### Secondary

- Couples and families
- Small teams (future)

## Jobs To Be Done

- Capture everything without friction
- Automatically plan day and week
- Coordinate with others effortlessly
- Maintain habits despite chaos
- Improve behavior over time

## Orbita Core

Based on the CORE framework:

- Capture
- Organize
- Review
- Execute

Features:

- Task capture (manual, chat, email)
- AI classification and prioritization
- Daily and weekly reviews
- AI time-blocking into calendars
- Multi-calendar sync
- CLI workflows

## Orbits (Modules)

- Smart Habits
- Smart 1:1 Scheduler
- Auto-Rescheduler
- AI Inbox Pro
- Priority Engine Pro
- Focus Mode Pro
- Ideal Week Designer
- Project AI Assistant
- Integrations Pack
- Time Insights
- Couples & Family Scheduler
- Automations Pro
- Wellness Sync

Each Orbit is independently subscribable.

## Success Metrics

### Product

- D30 retention > 35%
- Weekly active rate > 55%
- Scheduling automation > 70%

### Business

- ARPU €18–26
- Average modules/user ≥ 2.5
- Monthly churn < 4%

## Out of Scope

- Team collaboration (v1)
- Generic AI agent runtime
- Manual-heavy workflows
