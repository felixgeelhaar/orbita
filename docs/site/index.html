Task Management

Create, prioritize, and track tasks with intelligent scheduling. Automatic due date suggestions based on your patterns.

orbita task create "Ship feature" -p high
📅

Smart Scheduling

Auto-schedule tasks into optimal time blocks. Respects your energy levels and calendar constraints.

orbita schedule auto --optimize

Habit Tracking

Build consistent habits with streak tracking, adaptive scheduling, and completion logging.

orbita habit log "Morning workout"

Meeting Coordination

AI-Native with MCP Integration

Orbita speaks MCP (Model Context Protocol), enabling seamless integration with AI assistants like Claude. Let AI help manage your tasks, review your schedule, and provide productivity insights.

🔧 Full tool access for AI assistants
📋 Rich resources for context
💬 Pre-built workflow prompts

Get started in seconds

Install Orbita and start organizing your work

Homebrew

brew install felixgeelhaar/tap/orbita

Go Install

go install github.com/felixgeelhaar/orbita@latest

From Source

git clone https://github.com/felixgeelhaar/orbita && cd orbita && make build

Quick Start

Getting Started 
# Initialize Orbita
$ orbita init
✔ Configuration created at ~/.config/orbita/config.yaml

# Create your first task
$ orbita task create "Try Orbita" -p high
✔ Task created: Try Orbita

# See your dashboard
$ orbita dashboard

# Start the MCP server for AI integration
$ orbita mcp serve
ℹ  MCP server listening on :8080
