# orbita calendar & auth

Manage calendar connections, authentication, and sync.

## Authentication Commands

### orbita auth connect

Connect to a calendar provider.

```bash
orbita auth connect <provider> [flags]
```

**Providers:**
- `google` - Google Calendar (OAuth2)
- `microsoft` - Microsoft Outlook/365 (OAuth2)
- `apple` - Apple Calendar / iCloud (CalDAV with app-specific password)
- `caldav` - Generic CalDAV (Fastmail, Nextcloud, etc.)

**Flags:**
| Flag | Description |
|------|-------------|
| `--name` | Display name for the calendar |
| `--primary` | Set as primary calendar for imports |
| `--push` | Enable pushing Orbita blocks (default: true) |
| `--pull` | Enable pulling events from this calendar |
| `--url` | CalDAV server URL (required for caldav provider) |

**Examples:**

<Tabs>
  <TabItem label="Google">
    ```bash
    # Basic connection
    orbita auth connect google

    # Set as primary with pull enabled
    orbita auth connect google --primary --pull

    # Custom name
    orbita auth connect google --name "Work Calendar"
    ```
  </TabItem>
  <TabItem label="Microsoft">
    ```bash
    # Connect Microsoft 365
    orbita auth connect microsoft

    # Work account push-only
    orbita auth connect microsoft --name "Work" --push --no-pull
    ```
  </TabItem>
  <TabItem label="Apple">
    ```bash
    # Connect Apple iCloud
    orbita auth connect apple

    # You'll be prompted for:
    # - Apple ID (email)
    # - App-specific password
    ```
    <Aside type="note">
      Generate an app-specific password at [appleid.apple.com](https://appleid.apple.com/account/manage)
    </Aside>
  </TabItem>
  <TabItem label="CalDAV">
    ```bash
    # Fastmail
    orbita auth connect caldav --url https://caldav.fastmail.com

    # Nextcloud
    orbita auth connect caldav --url https://cloud.example.com/remote.php/dav

    # Custom CalDAV server
    orbita auth connect caldav --url https://caldav.example.com --name "Personal"
    ```
  </TabItem>
</Tabs>

### orbita auth list

List all connected calendars.

```bash
orbita auth list
```

**Output:**
```
Connected Calendars:
  ✓ google     felix@example.com      [primary] [push] [pull]
  ✓ microsoft  felix@company.com                [push]
  ✓ caldav     Personal (Fastmail)              [push]
```

---

### orbita auth disconnect

Disconnect a calendar provider.

```bash
orbita auth disconnect <provider>
```

**Examples:**
```bash
orbita auth disconnect microsoft
# Output: Disconnected Microsoft calendar.

orbita auth disconnect caldav
# Output: Disconnected CalDAV calendar.
```

### orbita calendar list

Check calendar connection and sync status.

```bash
orbita calendar status

  ✓ google     Last sync: 2 min ago
  ✓ microsoft  Last sync: 5 min ago
  ✓ caldav     Last sync: 3 min ago

Sync Status: Active
Next sync in: 2 minutes
```

---

### orbita calendar cleanup

Clean up sync issues.

```bash
orbita calendar cleanup [flags]
```

**Flags:**
| Flag | Description |
|------|-------------|
| `--duplicates` | Remove duplicate events |
| `--orphaned` | Remove orphaned sync records |

---

### orbita calendar reset

Reset calendar sync state.

```bash
orbita calendar reset [flags]
```

**Flags:**
| Flag | Description |
|------|-------------|
| `--provider` | Reset only specific provider |
| `--confirm` | Skip confirmation prompt |

<Aside type="caution">
  This removes all sync state. Events will be re-synced on the next sync cycle.
</Aside>

## Sync Commands

### orbita sync

Trigger calendar synchronization.

```bash
orbita sync [flags]
```

**Flags:**
| Flag | Description |
|------|-------------|
| `--all` | Sync to all connected calendars |
| `--provider` | Sync to specific provider only |
| `--status` | Show sync status |
| `--import` | Import external changes only |
| `--verbose` | Verbose output |

**Examples:**
```bash
# Sync to all calendars
orbita sync --all

# Sync to specific provider
orbita sync --provider google

# Check status only
orbita sync --status

# Import only (no push)
orbita sync --import

# Verbose sync
orbita sync --all --verbose
```

---

### orbita conflicts resolve

Resolve sync conflicts.

```bash
orbita conflicts resolve [id] [flags]
```

**Flags:**
| Flag | Description |
|------|-------------|
| `--auto` | Auto-resolve all conflicts |
| `--action` | Resolution action: `reschedule`, `keep`, `delete`, `ignore` |

**Examples:**
```bash
# Interactive resolution
orbita conflicts resolve

# Resolve specific conflict
orbita conflicts resolve 1 --action reschedule

# Auto-resolve all
orbita conflicts resolve --auto
```
