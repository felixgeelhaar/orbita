# 1:1 Meetings

Orbita helps you manage recurring 1:1 meetings with participants, talking points, and automatic preparation reminders.

## Creating Meetings

```bash
# Add a recurring 1:1
orbita meeting add "Weekly sync with Sarah" \
  --participant sarah@company.com \
  --frequency weekly \
  --duration 30

# Bi-weekly meeting
orbita meeting add "Manager 1:1" \
  --participant manager@company.com \
  --frequency biweekly \
  --day friday \
  --time "14:00"
```

## Managing Talking Points

```bash
# Add a topic
orbita meeting topic "Discuss Q1 goals" --meeting "Sarah"

# Add with priority
orbita meeting topic "Project timeline concerns" --meeting "Sarah" --priority high

# View topics
orbita meeting topics "Sarah"
```

## Preparing for Meetings

```bash
# See upcoming meeting with topics
orbita meeting prepare "Sarah"

# View all upcoming meetings
orbita meeting list --upcoming
```

Output:

