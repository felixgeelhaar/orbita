# Time Insights

Orbita tracks how you spend your time and provides actionable insights to improve productivity.

## Viewing Insights

```bash
# Today's summary
orbita insights today

# Weekly overview
orbita insights week

# Monthly report
orbita insights month
```

## Daily Insights

```bash
orbita insights today
```

## Weekly Trends

```bash
orbita insights week
```

Shows:
- Daily comparison
- Peak productivity hours
- Task completion rate
- Habit consistency

## Export Data

```bash
# Export to CSV
orbita insights export --format csv --output report.csv

# Export to JSON
orbita insights export --format json
```

## Focus Score

Orbita calculates a daily focus score based on:
- Deep work duration
- Context switch frequency
- Task completion rate
- Schedule adherence
