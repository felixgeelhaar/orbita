# Habit Tracking

Orbita helps you build and maintain habits with adaptive scheduling, streak tracking, and milestone celebrations.

## Creating Habits

```bash
# Basic habit
orbita habit create "Morning meditation"

# With frequency and duration
orbita habit create "Exercise" --frequency "3x/week" --duration 45

# With preferred time
orbita habit create "Read" --frequency daily --duration 30 --time "21:00"

# With specific days
orbita habit create "Weekly review" --frequency weekly --day friday --time "16:00"
```

### Habit Options

| Option | Description |
|--------|-------------|
| `--frequency` | `daily`, `weekly`, `Nx/week`, custom |
| `--duration` | Time in minutes |
| `--time` | Preferred time (HH:MM) |
| `--day` | Specific day(s) for weekly habits |
| `--reminder` | Enable reminders |

## Frequency Options

```bash
# Daily
orbita habit create "Journal" --frequency daily

# Weekly (specific day)
orbita habit create "Deep clean" --frequency weekly --day saturday

# Multiple times per week
orbita habit create "Gym" --frequency "3x/week"

# Custom pattern
orbita habit create "Water plants" --frequency "every 3 days"

# Weekdays only
orbita habit create "Morning standup" --frequency weekdays
```

## Logging Completions

```bash
# Log completion
orbita habit log meditation

# Log with note
orbita habit log exercise --note "30 min run"

# Log for specific date
orbita habit log reading --date yesterday

# Log multiple
orbita habit log meditation exercise reading
```

## Streak Tracking

### How Streaks Work

- **Daily habits**: Complete each day to maintain streak
- **Weekly habits**: Complete at least once per week
- **Nx/week habits**: Complete N times within the week

### Grace Period

Configure grace period for missed days:

```yaml
# ~/.orbita/config.yaml
habits:
  grace_period: 2  # Hours after preferred time
```

### Streak Recovery

If you miss a day:

```bash
# Check streak status
orbita habit streak meditation

# See what broke the streak
orbita habit history meditation --show-breaks
```

## Adaptive Scheduling

Orbita learns your patterns and adapts:

### Automatic Time Adjustment

If you consistently complete at a different time:

```
Originally scheduled: 07:00
Actual completions: 07:30, 07:25, 07:35, 07:40

Orbita suggests: Reschedule to 07:30?
```

### Frequency Adjustment

If you're struggling with frequency:

```
Target: Daily
Actual: 4-5 times per week

Orbita suggests: Reduce to 5x/week for more realistic goals?
```

## Managing Habits

### Update a Habit

```bash
# Change frequency
orbita habit update meditation --frequency "weekdays"

# Change preferred time
orbita habit update exercise --time "18:00"

# Change duration
orbita habit update reading --duration 45
```

### Pause a Habit

```bash
# Pause for vacation
orbita habit pause exercise --until 2025-02-01

# Resume early
orbita habit resume exercise
```

### Archive a Habit

```bash
# Archive (keeps history)
orbita habit archive old-habit

# View archived habits
orbita habit list --archived

# Restore archived habit
orbita habit restore old-habit
```

## Reminders

```bash
# Enable reminders
orbita habit update meditation --reminder

# Set reminder time (before scheduled time)
orbita habit update meditation --reminder-before 15

# Disable reminders
orbita habit update meditation --no-reminder
```

<Aside>
  Reminders require notification permissions on your system.
</Aside>

### Habit Blocks

Habits can be:
- **Flexible**: Scheduled when convenient
- **Fixed**: Scheduled at specific times

```bash
# Fixed time habit
orbita habit create "Standup" --frequency weekdays --time "09:00" --fixed

# Flexible habit (default)
orbita habit create "Reading" --frequency daily --duration 30
```
