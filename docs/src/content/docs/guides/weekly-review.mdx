# Weekly Review

The weekly review helps you stay aligned with your goals and adjust your approach.

## When to Review

- **Best time**: Friday afternoon or Sunday evening
- **Duration**: 30-60 minutes
- **Frequency**: Weekly

## Review Steps

### 1. Review the Past Week

```bash
orbita insights week
```

Look at:
- Tasks completed vs planned
- Habits consistency
- Time allocation

### 2. Clear Incomplete Tasks

```bash
orbita task list --status pending
```

For each pending task:
- Reschedule if still relevant
- Archive if no longer needed
- Break down if too large

### 3. Process Your Inbox

```bash
orbita inbox list
orbita inbox process
```

### 4. Review Habits

```bash
orbita habit list
```

Check streaks and adjust frequencies if needed.

### 5. Plan Next Week

```bash
orbita schedule week --next
```

Add major commitments and blocks.

### 6. Archive Completed

```bash
orbita task archive --completed
```

## Weekly Review Checklist

- [ ] Review insights
- [ ] Process inbox to zero
- [ ] Update pending tasks
- [ ] Check habit streaks
- [ ] Review calendar for next week
- [ ] Set 3 priorities for the week

## Tips

- Block time for the review itself
- Be honest about overcommitment
- Celebrate wins before diving into improvements
