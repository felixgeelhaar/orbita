Build Your Own Extensions

Orbita is fully extensible with two powerful SDK options

⚙

Engine SDK

Algorithms

Build custom scheduling, priority, and classification algorithms. Engines run as isolated gRPC plugins, enabling safe third-party algorithm marketplaces.

  • Scheduler Engines: Custom scheduling algorithms
  • Priority Engines: Task prioritization logic
  • Classifier Engines: Inbox categorization
  • Automation Engines: Rule evaluation
Engine Example 
// priority_engine.go
type EisenhowerEngine struct{}

func (e *EisenhowerEngine) CalculatePriority(
    ctx *sdk.ExecutionContext,
    input sdk.PriorityInput,
) (*sdk.PriorityOutput, error) {
    // Eisenhower matrix logic
    urgent := input.DueDate.Before(time.Now().Add(24*time.Hour))
    important := input.Tags.Contains("important")

    return &sdk.PriorityOutput{
        Score:    calculateScore(urgent, important),
        Quadrant: getQuadrant(urgent, important),
    }, nil
}

📦

Orbit SDK

Features

Create feature modules that extend Orbita's capabilities. Orbits run in-process with capability-restricted access to ensure safety and performance.

  • MCP Tools: Add new AI assistant tools
  • CLI Commands: Extend the command line
  • Event Handlers: React to domain events
  • Sandboxed Storage: Isolated data per orbit
Orbit Example 
// pomodoro_orbit.go
type PomodoroOrbit struct{}

func (o *PomodoroOrbit) RequiredCapabilities() []sdk.Capability {
    return []sdk.Capability{
        sdk.CapReadTasks,
        sdk.CapWriteStorage,
        sdk.CapRegisterTools,
    }
}

func (o *PomodoroOrbit) RegisterTools(reg sdk.ToolRegistry) error {
    return reg.RegisterTool("start_pomodoro",
        o.handleStartPomodoro, pomodoroSchema)
}

Get started in seconds

Install Orbita and start organizing your work

Homebrew

brew install felixgeelhaar/tap/orbita

Go Install

go install github.com/felixgeelhaar/orbita@latest

From Source

git clone https://github.com/felixgeelhaar/orbita && cd orbita && make build

Quick Start

Getting Started 
# Initialize Orbita
$ orbita init
✔ Configuration created at ~/.config/orbita/config.yaml

# Connect your calendar
$ orbita calendar connect google
✔ Connected to Google Calendar

# Create your first task
$ orbita task create "Try Orbita" -p high
✔ Task created: Try Orbita

# See your dashboard
$ orbita dashboard

# Enable an orbit
$ orbita orbit enable smart-habits
✔ Enabled: Smart Habits

# Start the MCP server for AI integration
$ orbita mcp serve
ℹ  MCP server listening on stdio
