# orbita inbox

Process items through the AI-powered inbox.

## Commands

### add

Add item to inbox.

```bash
orbita inbox add <text> [flags]
```

**Flags:**
| Flag | Description |
|------|-------------|
| `--source` | Source of item (slack, email, etc.) |

### list

List inbox items.

```bash
orbita inbox list
```

### process

Process inbox with AI assistance.

```bash
orbita inbox process [flags]
```

**Flags:**
| Flag | Description |
|------|-------------|
| `--auto` | Auto-process all items |
| `--interactive` | Interactive mode |

### review

Review AI suggestions.

```bash
orbita inbox review
```

### clear

Clear processed items.

```bash
orbita inbox clear
```
